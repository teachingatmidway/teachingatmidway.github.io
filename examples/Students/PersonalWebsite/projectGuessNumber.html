Guess the Number

"Guess the Number" is a simple interactive game where the user tries to guess a randomly generated number. Features include:

  • Helpful prompts indicating to guess higher or lower
  • Score tracking
  • History tracking
  • Confetti when you win

This project was developed using HTML, CSS, and JavaScript through React.
