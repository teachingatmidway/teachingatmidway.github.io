Calculator

"Calculator" is a responsive, user-friendly application designed to perform basic arithmetic operations. It provides a clean and intuitive interface, making calculations straightforward and efficient. Features include:

  • Basic Arithmetic: Perform addition, subtraction, and division
  • Responsive Design: Adapts seamlessly to diffrerent screen sizes

This project was developed using snack.expo.dev. It serves as an excellent demonstration of fundamental programming and UI/UX design principles.
